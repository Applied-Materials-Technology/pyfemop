# pyfemop


-----------------------------------------
PyFEMOP - Python Finite Element Model Optimisation Problems
-----------------------------------------
pyfemop is a python based tool for optimizing
FE models.
It will use the MOOSE FE package, with 
Gmsh to generate meshes. 



## Installation

```bash
$ pip install mtgo
```
